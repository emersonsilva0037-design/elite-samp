# Elite Samp Roleplay Gamemode Documentation

## Introduction
The Elite Samp Roleplay gamemode is designed to provide an immersive role-playing experience in the San Andreas Multiplayer (SAMP) environment. This document outlines the installation instructions, systems overview, and available admin commands.

## Installation Instructions
1. **Download the Gamemode:**  Clone the repository using the command:
   ```bash
   git clone https://github.com/emersonsilva0037-design/elite-samp.git
   ```

2. **Setup the Server:**  Ensure that you have a SAMP server set up. Install any dependencies needed for the gamemode.

3. **Add Gamemode to Server:**  Copy the `elite.samp` file into your server's gamemodes folder. Update the `server.cfg` file to use `elite` as the gamemode:
   ```plaintext
   gamemode0 elite
   ```

4. **Run the Server:**  Start your SAMP server and connect to it using the SAMP client.

## Systems Overview
- **Character Creation:** Players can create unique characters and customize their appearance.
- **Job System:** A comprehensive job system allows players to choose various career paths including police, medic, and more.
- **Economy System:** In-game currency and buying/selling mechanics allow players to engage in a vibrant economy.
- **Housing System:** Players can purchase and customize their homes.
- **Vehicle System:** A variety of vehicles are available, allowing players to travel around San Andreas efficiently.

## Admin Commands
Admins have access to a variety of commands to manage the server effectively. Below are some key commands:
- `/kick [player]` - Kicks a player from the server.
- `/ban [player]` - Bans a player from the server.
- `/mute [player]` - Mutes a player.
- `/tp [player]` - Teleports to a player's location.
- `/givecash [player] [amount]` - Gives in-game currency to a player.

For a complete list of commands, refer to the in-game admin panel.
